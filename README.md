# 🚚 Shipment Analytics Dashboard

## Problem
Lack of visibility into shipment delays, carrier performance, and route inefficiencies.

## Solution
Built an analytics platform with anomaly detection (Z-score & IQR) and an interactive dashboard.

## Features
- Shipment delay analysis
- Carrier performance tracking
- Route-level insights
- Anomaly detection
- Drill-down exploration
- Cascading filters

## Tech Stack
- Python
- Pandas
- Streamlit

## How to Run
pip install -r requirements.txt
streamlit run app.py
